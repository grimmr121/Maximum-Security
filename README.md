# Maximum-Security
ZDOOM DOOM 2 Mod

===========================================================================
Title                   : Maximum Security (WIP) v0.3b
Filename                : Maximum_Security.pk3
Advanced engine needed  : (GZDoom) g4.14.2
Primary purpose         : Single player, coop can work but not perfect
Author                  : GrimmR (Doomworld)
Misc. Author Info       : Your local neighbourhood mapper. This is my first map set uploaded, I am open to critiques and suggestions please.

Theme			: Maps with puzzle/horror elements. Extraplanar travel. Focus on unique boss battles.

Description             : Beneath the prison, something stirs. In order to escape, you will have to awaken it. Doomguy is going to have to 
			: use his brains as well as his fists if he wants to survive this one!
			: The zombification of all base personnel is just Tuesday for Doomguy, he is used to that.
                        : No, there is something darker going on here, and a mysterious extra-dimensional gate   
                        : beckons from beneath the Earth. Where does it lead? Maybe you should have stayed in your cell.

**Installation**
Extract 
"Max_Sec_Standard.bat"
and "mods"
folder to your GZDOOM folder. Make sure DOOM2.WAD is in the same folder.
Double-click "Max_Sec_Standard.bat" and play!
